# Virtual Environment Setup

## Location
Virtual environment created at: `D:\venvs\stock`

## Creation Command
```bash
python -m venv "D:\venvs\stock"
```

## Activation (Windows)
```cmd
D:\venvs\stock\Scripts\activate
```

## Activation (PowerShell)
```powershell
D:\venvs\stock\Scripts\Activate.ps1
```

## Deactivation
```bash
deactivate
```

## Notes
- This virtual environment must be activated before running any Python commands for this project
- All dependencies should be installed within this activated environment
- The virtual environment contains Python 3.11+ as required by the project dependencies
